SS.LV Auto WebScraper

2024-06-12

A tool what fetches data such as descriptions, links, models, years, capacities, mileage, and prices from multiple pages and exports it to a CSV file.

Features

  • Fetches car listings from ss.lv
  • Extracts detailed information including description, link, model, year, engine capacity, mileage, and price
  • Handles multiple pages of listings
  • Exports data to a CSV file

Dependencies

The code is written in Python and utilizes the following libraries:

  • requests
  • beautifulsoup4
  • pandas
